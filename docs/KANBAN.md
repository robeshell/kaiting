## 当前焦点

**SND-102 — 跨平台本地文件夹持久权限**

SQLite 资料库和原子扫描事务已经完成。下一步建立真正可持久化的目录访问：
macOS 使用 security-scoped bookmark，Android 使用 SAF tree permission，
Windows 恢复用户选择的目录；权限失效必须成为可恢复状态。

## 第一版门槛

开始更大范围的视觉或播放列表工作前，以下项目必须全部完成：

- 一个真实本地 MP3 和 FLAC 可在 Android、Windows 被索引、持久化、展示和播放。
- 认证 WebDAV 索引与 Range 播放在 Android、Windows 通过。
- 队列切换和保存进度不会造成进度回退。
- Android 后台控制与 Windows 系统媒体控制保持同步。
- 目录权限撤销、服务器不可用、凭据错误和音频损坏均有明确恢复状态。

相关证据：[播放验证](PLAYBACK_VALIDATION.md)；[架构](ARCHITECTURE.md)。
