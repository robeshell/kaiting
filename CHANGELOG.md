## [1.0.4] - 2026-07-19

版本号：`1.0.4+7`

### Added

- **搜索**
  - 支持中文**拼音全拼**与**首字母**匹配（如 `yanzi`、`syz`）
  - 结果分区：顶部匹配**艺人**、**专辑**，下方为歌曲列表
  - 字段筛选文案改为「匹配范围」（歌名 / 专辑名等），避免误解为实体类型
  - 空状态展示最近搜索；结果超过 200 首时提示截断
  - 支持将搜索结果作为队列播放
- **导航**
  - 艺人、专辑名在搜索、正在播放、迷你播放器、专辑详情、资料库歌曲、收藏 / 最近、播放列表、艺人 / 流派集合、播放队列等处可点进对应页
  - 可点元数据保持与原先副标题一致的视觉样式（不加粗、不改色、无下划线）
- **正在播放**
  - 播放器样式精简为「经典」与「黑胶」；旧布局映射到经典
  - 独立选项：默认打开歌词（手机端）
  - 黑胶：页面未激活或系统「减少动态效果」时停止旋转

### Fixed

- **资料库 / 封面**
  - 扫描时标签优先：大图或残缺 `PICTURE` 不再导致标题 / 艺人变成「未知」
  - 拒绝写入残缺封面缓存；修复大图 PNG 完整性校验误判（`IEND`）
  - WebDAV / FLAC 渐进读取封面更可靠
- **播放**
  - 重启或恢复会话后，队列中非当前曲的歌词从资料库重新挂载
- **界面**
  - 桌面迷你播放器顶部分割线、阴影与进度条发丝线
  - 缺失封面时不再弹出红色 `PathNotFound` 预缓存错误条
  - 桌面内容区为底部播放条预留间距
  - 移动端列表中点击艺人 / 专辑名可正确跳转（不再被整行「单击播放」手势吞掉）

## [1.0.0] - 2026-07

对应标签：`v1.0.0`  
跨平台音乐播放器的首次公开可用版本。

### Added

#### 平台与播放

- Flutter 跨端壳：Android、iOS/iPadOS、macOS、Windows；Web 预览
- 基于 `just_audio` 的本地播放（MP3、FLAC 等，后续扩展多格式）
- WebDAV 远程源：认证、字节范围寻址、后台播放基础
- 无缝切歌（gapless）与播放模式（顺序 / 循环 / 单曲 / 随机）
- 可编辑播放队列；会话位置检查点与安全恢复（不自动续播）
- 播放状态展示一致化；迷你播放器可完整交互
- 歌词与播放进度同步

#### 资料库

- Drift / SQLite 持久化资料库（本地索引 + 用户状态）
- 本地文件夹权限与扫描；增量扫描
- WebDAV 扫描与增量扫描
- 专辑 / 艺人 / 流派 / 歌曲浏览；多碟与专辑分组
- 排序与来源筛选（本地 / WebDAV）
- 收藏、最近播放、完整播放历史
- 可编辑播放列表（增删改、排序、下一首播放）
- 资料库搜索（字段筛选与排序，大库 isolate 检索）
- 离线下载与 provider 架构；播放失败恢复与应用工具入口

#### 体验与桌面

- 封面优先界面：资料库、专辑详情、正在播放、响应式迷你播放器
- 全局组件与主题统一（Sound 视觉语言）
- 桌面：单击选中、双击播放；键盘导航与全局快捷键（含 `Cmd/Ctrl + /` 说明）
- 短窗口 / 低高度布局适配；大库滚动与封面性能优化

#### 工程

- 发布脚本 `tool/release.dart`（版本 bump + 多平台产物到 `dist/`）
- 性能基线与看板 / 架构文档

### Fixed

- 启动零尺寸视口崩溃防护
- 空格快捷键与文本输入冲突
- 其他早期壳层与播放回归问题
